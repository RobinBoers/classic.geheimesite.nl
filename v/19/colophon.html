Colophon

This site was handcrafted using Neovim, the perfect hammer for an imperfect world.

Everything on this site is released into the public domain. The source code is also publicly available and licensed AGPLv3.

I spent a lot of time and effort tweaking the server configuration. It currently does compression, caching, CSP and a bunch of privacy-related headers, because the modern web absolutely sucks by default, and I prefer visitors to not be tracked by their own browsers.

For a full breakdown of how my site was build, see the README.
