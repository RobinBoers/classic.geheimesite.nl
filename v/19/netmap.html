Netmap

Most sites have some sort of Sitemap. A big document or list containing a bunch of links to a bunch of pages on the site. If you’re looking for my sitemap, well sucks to be you, because this is not it.

But Robin, what is this then? This is a netmap. It’s not a collection of pages on a site, but a collection of sites in a network. My network, to be exact. And in case you were wondering, yup, I made up the word :)

If you are looking for a sitemap, it’s right here.

Active

  • Roblog (this site)
  • dupunkto.org: a pubnix/tilde that I’m co-maintaining.
  • git.dupunkto.org: git hosting for dupunkto members.
  • ftp.dupunkto.org: file sharing for dupunkto members.
  • school.geheimesite.nl: a website with a bunch of study materials I made for my classmates.
  • classic.geheimesite.nl: a public archive of older versions of this website; a private Wayback machine if you will.
  • sharefox.geheimesite.nl: my public Sharefox inbox for receiving encrypted messages.

Archived

  • blog.geheimesite.nl: public archive of all content on my old (Blogger) blog, from before I migrated the posts to my own site.
  • micro.geheimesite.nl: public archive of my short-lived microblog running on my own DIY blogging software called neopub.
  • nindo.geheimesite.nl: public archive of all Nindo content posted before I killed the project.

Defunct

  • geheimesite.nl: this used to be my personal site; first launched in november 2017 when I was in primary school.
  • code.geheimesite.nl: I used to put random projects that required hosting on this domain.
  • git.geheimesite.nl: former selfhosted Gitea-instance, replaced by {du}punkto; currently serves a redirect.
  • matrix.geheimesite.nl: former selfhosted Matrix homeserver; I’ve since moved to IRC.
  • mc.geheimesite.nl: hosted multiple Minecraft servers before I took it down.
  • chat.geheimesite.nl: former official instance of my ThinkChat project.
  • {demo, pages, docs}.geheimesite.nl: domains used for GitHub pages.
