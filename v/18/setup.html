Linux Dev setup

My setup for software/web development on Linux. Elixir, NodeJS, Python, PostgreSQL and PHP.

üêß Distro

My current distro of choice is NixOS. I like the immutable and reproducable system it provides, and the atomic upgrades are a nice bonus. I mainly use a tiling window manager called Sway, which is a wayland compositer. I like wayland because I find it to be way more responsive/smooth than Xorg. I integrated Sway with gnome services, similar to the sway-gnome fork that I used on Arch.

You can find my entire Nix configuration in my dotfiles repository.

ü¶ä Webbrowsers

For most of my webbrowsing needs I use Firefox. I love customizing my browser using custom CSS. Currently I’m rocking vertical tabs, which I stole from ranmaru22/firefox-vertical-tabs.

I also use (Ungoogled) Chromium to test my projects in another browser and if I need to use some Chrome-only app or API.

See my firefox-tweaks repo for my current Firefox setup.

Browser extensions

  • Dark reader: dark mode for websites that don’t support it out-of-the-box.
  • Don’t track me Google: removes tracking info from links.
  • DuckDuckGo Privacy Essentials: see tracking and privacy info for sites.
  • Enhancer for YouTube: remove ads and get cool features.
  • Enhanced GitHub: extra features for GitHub like repo size and download buttons.
  • Facebook Container: uses container tabs to isolate Facebook tracking.
  • Firefox Multi-Account Containers: containers for work/school related accounts.
  • uBlock origin: blocks ads :)
  • Web Archives: see earlier versions of sites.
  • I still don’t care about cookies: gets rid of cookie popups automatically.

VSCode also syncs my setup across machines using settings sync with my GitHub account.

Theming

I like my editor, like most programmers, to be dark. My favorite dark themes are:

  • Adwaita. It integrates nicely with my other GTK applications.
  • Frontend Delight Dark. Ihis is a colorscheme that I ported from somewhere else, but I can’t quite remember where I found it.
  • GitHub Dark. Ihis is the old GitHub dark theme for VSCode. The new one is called GitHub Dark Default.

I also like Ayu dark, Tokyo Night and Adwaita Light, but rarely use light mode. For file icons is use vscode-icons, and for product icons I use Fluent icons.

My favorite bitmap font is Terminus. In other applications I use Jetbrains Mono because it looks clean and has support for fancy ligatures.

I used to use DejaVu Sans Mono and a fork which added ligatures support, called DejaVu Sans Code.
