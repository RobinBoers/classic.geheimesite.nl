Over deze site

Mijn site is gebouwd met mijn eigen static site generator, genaamd Chop. Hij wordt gehost bij bHosted. Alle inhoud waarvan ik de auteur ben is in het publieke domein (specifiek CC0) De source code is openbaar beschikbaar onder de AGPLv3.

De site heeft een grote focus op eenvoud, toegankelijkheid en duurzaamheid. Ik schrijf in kale Markdown, die daarna wordt omgezet in HTML en Gemtext door een klein programmaatje genaamd Chop. Tijdens dit proces worden afbeeldingen verkleind naar 600x en worden de ~400 regels ambachtelijke CSS geminified. Met uitzondering van de achtergrond gebruikt de site het colorscheme en de fonts die in de browser zijn ingesteld.

De server is zo ingesteld dat pagina’s worden gecompressed en verstuurd met correcte caching headers. Ook maak ik gebruik van een strenge CSP, die scripts en onveilige content blokkeert. Ik stuur ook een aantal privacy-gerelateerde headers mee, want het web is een complete shitshow by default, en ik heb liever niet dat bezoekers door hun eigen browser worden getrackt.
